| Name              | Description                                                                                             | Stacks
| ----------------- | ------------------------------------------------------------------------------------------------------- | ------ |
| [real_estate](https://github.com/gorbatenkomv/Portfolio/tree/main/real_estate) | Development model for market value of real estate determination. | Pandas |
